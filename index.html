ToyORB - BrokerPattern

ToyORB este un Object Request Broker (ORB) personalizat care permite comunicarea distribuită orientată pe obiecte între clienți și servere, fără a utiliza tehnologiile middleware existente, cum ar fi Java RMI sau gRPC. Acesta acceptă apeluri de procedură la distanță (RPC) în diferite procese, limbaje și platforme.
Tehnologii utilizate: Java, reflection, javascript, C++, stilul arhitectural Broker.
